# 🎓 Student Performance Predictor

A Machine Learning web application that predicts a student's overall performance and estimated average score based on background factors.

🚀 **Live App:** https://student-performance-predictor-wbek.onrender.com/

---

## 📌 Project Overview

This project uses a trained Machine Learning model to analyze student-related factors such as gender, parental education, lunch type, and test preparation.

Based on these inputs, the system predicts:
- 📊 Estimated Average Score
- 🎯 Performance Level (Good / Average / Poor)

---

## 🧠 Technologies Used

- Python 🐍
- Flask 🌐
- Scikit-learn 🤖
- NumPy 📊
- HTML/CSS 🎨
- Gunicorn 🚀 (for deployment)
- Render ☁️ (hosting)

---

## ⚙️ Features

- Simple and user-friendly interface  
- Real-time prediction  
- Performance classification  
- Deployed as a live web application  
- Works directly in browser (no installation needed)

---

## 🧾 Input Factors

The model uses the following inputs:

- Gender  
- Student Category  
- Parental Level of Education  
- Lunch Type  
- Test Preparation Course  

---

## 📤 Output

- 🎯 Performance Level (Good / Average / Poor)  
- 📊 Estimated Average Score

## 💻 How to Run Locally

1. Clone the repository:
git clone https://github.com/your-username/student-performance-predictor.git

2. Navigate to the project folder:
cd student-performance-predictor

3. Install dependencies:
pip install -r requirements.txt

4. Run the application:
python app.py

5. Open in browser:
http://127.0.0.1:5000/

## 🌐 Deployment

The application is deployed using **Render**.

- Backend: Flask  
- Server: Gunicorn  
- Hosting: Render Cloud
